# Smart Light Control System

## Project Overview
This project demonstrates a simple smart light control system using Arduino UNO and LED simulation in Tinkercad.

## Components Used
- Arduino UNO
- LED
- 220Ω Resistor
- Jumper Wires

## Working Principle
The Arduino receives serial commands through the Serial Monitor:
- '1' → LED ON
- '0' → LED OFF

## Software Used
- Tinkercad Circuits
- Arduino IDE

## Features
- Simple ON/OFF light control
- Beginner-friendly project
- Simulated hardware environment
- Real-time serial communication

## Applications
- Home Automation
- Smart Lighting Systems
- Embedded Systems Learning
- IoT Basics

## Mobile App Simulation
In this project, Serial Monitor commands are used to simulate app-based smart light control functionality in a virtual environment.
